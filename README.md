# Gataria Backend Catalog

Variáveis de ambiente necessárias:
- MONGO_DATABASE
- MONGO_URL

Exemplo:

```
export MONGO_URL="mongodb+srv://gataria-tg:minhamae@cluster0.djihq0c.mongodb.net/"
export MONGO_DATABASE="dev"
```

Iniciando o backend

```
npm start
```

Para rodar os testes

```
npm test
```
